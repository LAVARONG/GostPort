# 更新日志 (Changelog)

所有针对平台后端的 API 更新、前端 UI 重构或重大 Bug 修复都将详细记录于此。遵循倒序时间轴排版规范。

## [2026-04-01] - 底层协议智能重构与旧版平滑升级 🛡️

- **Windows 防火墙自动化绑定**：为了打通本地端口暴露在外网时的环境壁垒，现已重写了 `manager.go` 中的底端进程生命周期挂载流程：
  1. **伴生开启**：当 Gost 管道启用时，程序会自动基于操作系统的 `netsh advfirewall` 构建匹配协议 (TCP/UDP) 和端口的隐式 Inbound 许可放行规则，避免被本地拦截导致外网或测速失联。
  2. **智能回收**：在用户手动停止转发规则、底层管道异常崩溃断开或服务主进程 `StopAll` 被切断时，均会全自动通过精确组装的 Rule Name 定点猎杀残存的防火墙暴露条目，绝不给您的设备系统留下哪怕一点冗余垃圾，完全恪守安全底线。
- **旧配置兼容容灾**：程序启动读取 `config.json` 加载映射配置时会开启无感升级机制，对于缺少 `protocol` 参数的老版本历史条目和外部直连调用，现在会自动初始化挂载为 `TCP+UDP` 双重并联模式并自动将更新持久化，彻底杜绝向下兼容隐患。

## [2026-04-01] - 核心增强：多协议转发支持 (TCP / UDP / TCP+UDP) 🚀

- **全协议兼容**：突破了前期仅支持单一 TCP 转发的限制，原生集成 UDP 映射能力。
- **底层架构升级**：
  1. 重构下发引擎，精准构建 `udp://` 及多路复用的 `-L` 参数管道。
  2. 实现 `TCP+UDP` 模式，通过单张前端规则即可自动拆解并联启动底层的双重监听并实现端口复用转发。
- **界面交互适配**：在面板的创建与编辑模块中加入直观的协议类型选择器，并在项别列表层无缝嵌入极简提示角标，在不破坏架构留白美感的前提下提供清晰的连接状态感知。

## [2026-03-31] - 体验升级：Windows Server 专属环境部署支持 🛠️

- **一键环境封装**：官方正式下发 PowerShell 工具箱 `setup.ps1`，彻底针对系统拦截等环境痛点进行降维打击，遵循核心逻辑：“关杀毒才能下 gost，开端口映射能用”。
- **架构级特性**：
  1. **彻底阻断 Defender**：调用原生组件深度注销一切微软防病毒服务和右键注册表项。
  2. **端点开放**：针对大批量或多范围转发管理需求，内置极简交互式的防火墙端口放行（支持 `10001-10020` 等连续端口及 TCP/UDP 矩阵）。
  3. **边缘附加**：附赠静默 Edge 的无干预部署指令。
- **文档升级**：全面同步至根目录 `README.md`，提供使用指引。

## [2026-03-24] - 体验升级：外网原生呈现与全场景一键复制 ✨

- **配置分离显示**：彻底解决监听 `0.0.0.0` 时地址无法点击直接使用的问题，在 `.env` 中引入 `EXTERNAL_IP`。前端现在会全自动对等替换展示真实的大网 IP。
- **效率交互增强**：抛弃繁琐的手动框选，在管理面板上实现了双侧 IP 和端口的「沉浸式一键复制」，通过精确拦截 `$event.currentTarget` 实现局部的、优雅的『已复制』反馈动效，最大程度提升运维效率。

## [2026-03-24] - 安全加固：外网支持与全栈 BasicAuth 🛡️

- **网络与权限**：
  1. 引入 `.env` 支持，开放 `WEB_HOST` 和 `WEB_PORT` 变量进行底层注入，可自由将环境绑定到 `0.0.0.0`，在默认保障 `127.0.0.1` 的克制隔离外提供选择。
  2. 实现内核级 BasicAuth 中间件全路由接管，若设置了 `WEB_USERNAME` 和 `WEB_PASSWORD` 环境变量，则严苛阻断越权访问。
- **配置一致性**：
  无感前向兼容原有阵列版 `config.json`，在提升安全水位的过程中丝毫未侵入极简的 JSON 结构规范。

## [2026-03-24] - 体验升级：底端规则锁定重排与全量测速引擎 ⚡

- **底层强化**：
  1. 重写后端规则下发流：强行按数组 `LocalPort`（本地端口）进行稳态升序重置，彻底解决前期因 Go map 无序导致面板规则列表上下乱跳的疑难杂症。
  2. 融入微秒级 TCP / IP 握手测时机制：在 `main.go` 暴露高并发 `/api/rules/ping` 接口，基于 `net.DialTimeout` 与 IPv6 标准库安全检测远端握手耗时。
- **界面调优**：
  1. 每条映射项增设防状态剥离的**隐秘单测点按键** ⚡，无侵入式注入防页面局部渲染剥夺的 UI 组件 `latencyMap`，优雅显示实时延迟或超时告警。
  2. 顶部主页眉区植入**「全部测速」**按钮，点击瞬间下发并行嗅探。
